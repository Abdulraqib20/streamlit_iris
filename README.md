# streamlit_iris
In this project, Imade use of streamlit to build a Machine Learning application that is capable of predicting the class of species of plants in the infamous `iris` dataset.
## The packages I used includes:
  - streamlit == 1.15.0
  - pandas == 1.4.2
  - numpy == 1.23.4
  - scikit-learn
  - plotly == 5.11.0
